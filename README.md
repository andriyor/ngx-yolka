# ngx-yolka

[![version][version-badge]][package]

Order according to fucking "yolochka" style

## Installation

``npm i -g ngx-yolka``

## Usage 

Run following command at root of project

``tso``

```bash
tso --help                                    

  Examples
    $ node tso
    $ node tso --file examples/search-input.module.ts
```
